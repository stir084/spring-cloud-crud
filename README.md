# MSA 환경에서의 CRUD 게시판   

MSA 환경 실습을 위해 만들어 본 CRUD 게시판 입니다.   
 
## 기술 스택
- Spring Cloud
- Kotlin
- Mysql
- Vue 2
- Armeria(gRPC) - 개발 예정
